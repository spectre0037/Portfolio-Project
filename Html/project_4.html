Automated MCQ Generator

A tool that extracts text from PDFs and generates MCQs using AI and NLP.

Key Features

  • Uploads PDFs and extracts key content.
  • Uses OpenAI API to generate MCQs.
  • Exports MCQs in CSV format.

How It Works

  1. Upload your PDF document.
  2. The tool extracts the text content from the PDF.
  3. Using AI and NLP, the tool generates multiple-choice questions.
  4. Download the generated MCQs in CSV format.
